# 개인 맞춤형 주식 매매 일지 (Trading Journal) 개발 계획서

## 1. 프로젝트 개요
사용자의 매매 기록을 체계적으로 관리하고, 손익을 분석하여 트레이딩 실력을 향상시키기 위한 웹 애플리케이션입니다.

## 2. 기술 스택 (Tech Stack)
### Backend
- **Language**: Kotlin (JDK 21)
- **Framework**: Spring Boot 3.x
- **Database**: H2 (개발용) -> MySQL (운영용)
- **ORM**: Spring Data JPA
- **Build Tool**: Gradle

### Frontend
- **Language**: TypeScript
- **Framework**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Library**: shadcn/ui 또는 MUI
- **HTTP Client**: Axios or Fetch

## 3. 아키텍처 (Architecture)
- **Client-Server 분리형 구조**
    - **Backend**: 8080 포트 (API Server)
    - **Frontend**: 5173 포트 (UI Client)
    - **Proxy**: 개발 환경에서 Frontend -> Backend API 호출 시 Proxy 설정

## 4. 핵심 기능 (MVP)
### 1) 매매 기록 입력 (POST)
- 종목명, 매수/매도 단가, 수량, 수수료, 진입 이유 기록
- `Trade` 엔티티 설계 및 API 구현

### 2) 손익 계산 및 조회 (GET)
- 실현 손익(Realized P&L) 자동 계산
- 일별/월별 수익률 리스트 조회

### 3) 대시보드 차트
- 누적 수익 곡선 시각화 (Recharts/Chart.js 활용)

## 5. 개발 단계 (Milestones)
1.  **프로젝트 초기화**: Spring Boot & React 프로젝트 생성 및 연동
2.  **백엔드 API 개발**: Trade 엔티티, Repository, Service, Controller 구현
3.  **프론트엔드 UI 개발**: 입력 폼, 리스트 뷰, 대시보드 구현
4.  **통합 및 테스트**: E2E 테스트 및 버그 수정
